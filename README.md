# Sales Dashboard - Tableau

## Project Overview
This project is an interactive Tableau dashboard created to analyze vehicle sales data. The dashboard helps visualize sales trends, manufacturer-wise performance, and key patterns using charts and filters.

## Tools Used
- Tableau
- Data Visualization
- Data Analysis
- Dashboard Design

## Features
- Interactive dashboard for vehicle sales analysis
- Manufacturer-wise sales comparison
- Vehicle type and horsepower-based filtering
- Time-based trend visualization
- Clean and user-friendly visual layout

## Key Insights
- Identified sales trends across different manufacturers
- Compared vehicle performance by type and horsepower
- Created visual reports for better decision-making
